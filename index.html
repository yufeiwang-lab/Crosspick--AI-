关于 CrossPick

产品定位

CrossPick 是一款面向跨境电商卖家的 AI 选品决策助手。通过整合多平台市场数据，运用 AI 大模型进行多维度分析，帮助卖家快速识别高潜力商品、规避选品风险、优化利润结构。

技术方案

前端

React + TailwindCSS + Chart.js

AI 能力

OpenAI GPT-4 + Coze Agent

数据源

Amazon API / 1688 API / Google Trends

部署

Vercel Edge + Supabase

产品亮点

  • •30秒完成一个品类的多维度选品分析
  • •AI自动生成利润测算与定价建议
  • •基于竞品数据的差异化策略推荐
  • •支持多平台（Amazon/速卖通/1688）数据整合
  • •实时趋势预警与季节性波动提醒
